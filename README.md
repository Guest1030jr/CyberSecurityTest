# Тест за дигитална безопасност
## Какво представлява?

Това е конзолно приложение на C#, което прави кратък тест за дигитална безопасност. Отговаряш на въпроси, свързани с онлайн рискове, и накрая виждаш своя резултат.

## Как работи?

- Програмата показва въпрос

- Имаш 4 възможни отговора (A, B, C, D)

- Избираш един отговор

- Получаваш информация дали е верен

- Накрая се показва общият резултат

## Какво използва?

- Списък с въпроси

- Списък с възможни отговори

- Списък с верните отговори

- Брояч на точки
## Цел на проекта

Да помогне на учениците да научат повече за безопасното поведение в интернет по забавен и лесен начин.
